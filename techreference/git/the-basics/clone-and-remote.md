# Clone and Remote

## Commands

### Clone

```
git clone <urlToRemote>
```

Clone under a different folder

```
git clone <urlToRemote> <diffFolderName>
```

### Push

```
git push
```

### Remote

Displays information on which remote is used for fetch and push

#### Basic

```
git remote
```

#### Detailed

```
git remove -v
```
