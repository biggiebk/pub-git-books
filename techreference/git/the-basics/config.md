# Config

## Commands

### Alias

```
git config (--global) alias.<aliasName> "<command> (arg1) (arg2) (...)"
```

### List

```
git config -l
```

#### With Origins

```
git config -l --show-origin
```

### Unset

```
git config (--global) --unset <keyName>
```

### Update

```
git config --global <key> <value>
```
